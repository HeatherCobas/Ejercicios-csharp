Ejercicio 10 - Act1: Inicio de Sesión
Descripción
Esta actividad consistió en desarrollar un sistema de inicio de sesión funcional con Windows Forms y arquitectura por capas. El sistema permite la autenticación de usuarios con validaciones, manejo de roles y restricciones específicas según el rol asignado.

Además, incluí una funcionalidad para que los usuarios puedan enviar mensajes al equipo de soporte y para que el soporte pueda gestionar dichos mensajes. Sin embargo, esta funcionalidad aún no estaba totalmente operativa en esta versión.

Funcionalidades destacadas
Validación de campos para usuario y contraseña.

Soporte para múltiples roles y restricciones basadas en ellos.

Funcionalidad inicial de mensajes entre usuarios y soporte (en desarrollo).

Mensajes de error claros para credenciales incorrectas o accesos no permitidos.
